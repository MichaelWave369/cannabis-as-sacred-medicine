# Changelog

All notable changes to this public navigator and release package will be documented here.

This project uses two related version tracks:

- **Paper/content release:** the underlying public paper and companion materials.
- **Site polish release:** improvements to the React navigator, metadata, accessibility, and public presentation.

## v1.0.2 Site Polish — 2026-05-27

### Added

- Root-level `CHANGELOG.md` to make future updates transparent.
- Open Graph and Twitter/X preview metadata for better link sharing.
- Mobile-friendly horizontal section navigation.
- “How to cite” box on the public navigator.
- Expanded footer with version, dual-license notice, citation guidance, and disclaimer.

### Changed

- Updated site version label to distinguish the navigator polish release from the v1.0.1 paper/content release.
- Updated downloads section wording now that the release PDFs are live.
- Improved public-launch readiness without changing the paper thesis or claim-boundary architecture.

### Preserved

- Claim discipline: clinical evidence, ritual history, contested interpretation, ecology, and testimony remain separate categories.
- License split: MIT for code; CC BY-NC-SA 4.0 for written materials, testimony language, diagrams, PDFs, and visual assets.
