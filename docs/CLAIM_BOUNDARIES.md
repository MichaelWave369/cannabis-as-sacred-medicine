# Claim Boundaries

The power of this project is that it argues for reverence **without pretending reverence is evidence**.

## Core discipline

- Clinical evidence stays clinical.
- Ritual history stays historical.
- Qaneh-bosem stays contested.
- Bees, hemp, and honey stay ecological-symbolic.
- Lived testimony stays testimony.
- Sacred medicine stays ethical synthesis.

## Sacred medicine definition

In this project, **sacred medicine** means:

> A biologically active medicine held in a container of reverence, evidence, consent, maturity, cultural humility, and harm-reduction boundaries.

Sacred is used here as an **ethical category**, not a scientific measurement or claim of divine proof.

## Safe-language examples

Instead of: **Cannabis cures pain.**  
Say: Cannabis or cannabinoids may help some adults with chronic pain; benefits vary by person and condition.

Instead of: **Cannabis is harmless because it is natural.**  
Say: Cannabis is powerful enough to help and powerful enough to harm; it deserves respect and boundaries.

Instead of: **Cannabis was definitely in the Bible.**  
Say: Some argue qaneh-bosem may refer to cannabis, but the identification remains contested.

Instead of: **Cannabis should replace opioids.**  
Say: Some patients report substituting cannabis for opioids, but opioid changes should be medically supervised.

## What this project does not do

This project does not provide dosing protocols, individual medical advice, legal advice, spiritual direction, a systematic review, or a theological proof.
