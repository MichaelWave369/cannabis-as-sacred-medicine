# Launch Checklist

Status: **Soft public release first. Official launch after verification.**

## Before launch

- [ ] Upload PDFs to `public/papers/`.
- [ ] Upload images to `public/images/`.
- [ ] Run `npm install`.
- [ ] Run `npm run build`.
- [ ] Confirm all download links work locally.
- [ ] Confirm mobile layout on phone width.
- [ ] Confirm keyboard navigation and focus states.
- [ ] Enable GitHub Pages using GitHub Actions.
- [ ] Confirm live URL loads.

## Soft launch

- [ ] Share with trusted reviewers first.
- [ ] Ask reviewers to check links, mobile layout, and wording boundaries.
- [ ] Fix any obvious issues.

## Official launch

- [ ] Share main landing page.
- [ ] Share executive summary.
- [ ] Share policy/testimony brief where appropriate.
- [ ] Share safe-language guide as standalone resource.
- [ ] Keep tone evidence-informed, dignified, and bounded.

## Do not

- Do not claim cannabis is harmless.
- Do not claim cannabis cures all pain.
- Do not claim qaneh-bosem is settled proof.
- Do not use the author's testimony to imply product endorsement.
- Do not present the paper as a medical guideline.
